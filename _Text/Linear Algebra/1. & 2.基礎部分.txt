基礎部分
	矩陣相乘
		c(i,j) = ∑ a(i,k)*b(k,j)
		沒有交換律
	trace
		tr(AB) = tr(BA)
		tr[(AB)C] = tr[C(AB)]
		兩兩可互換，不要亂換
	可逆(下列都是雙向)
		Ax = 0 只有零解
			ker(A) = { 0 }
			nullity = 0
		nonsingular
		det(A) ≠ 0
		沒有 0 這個特徵根
			det(A) = 特徵根相乘
			tr(A) = 特徵根相加
		Ax = b 具唯一解
			行獨立
	列運算
		列互換
			行列式值多負號
		列乘積
			行列式值多乘積
		列乘積和
			行列式值不變
		做列運算等同左乘一個可逆的列基本矩陣
	LU 分解
		做高斯消去，步驟寫好
			做到上三角矩陣後，得 U
			把剛剛做的第三型列運算稍做處理
				列運算 = 左乘可逆的列基本矩陣
			最後可整理出 L*U 的形式
		Ax = b
			Ax = LUx = b
			令 Ux = y，先解 Ly = b
			最後算 Ux = y，求得 x
	古典伴隨矩陣
	classical adjoint matrix
		adj(a(i,j)) = 對 j,i 項取餘因子
		餘因子：類似降階手法、去除該行&列後求行列式值
		if A 可逆，則
			A ( adj(A)/det(A) ) = I
			A inverse = [1/det(A)] * adj(A)
	其他工具
		方塊矩陣
			矩陣相乘、行列式等均可以類似方式運算
			將方塊矩陣視為一個元素去處理即可
		Cramer’s rule
			對有解的線性系統而言
				取 n+1 個行列式值可得系統的解
				n 個為 Ax = b 中，A 的行以 b 分別替換後求行列式值
				1 個為 A 本身的行列式值
			能用的地方不多、少考
		Wronskian
			對 function f1, f2, f3, …, fn，每個都 n-1 次可微分
			排成列，第 i 列為 i-1 次微分
			定義 W(x) 為上述排列而成的行列式值
				如果存在 x 使得 W(x) ≠ 0
					f1, f2, f3, …, fn 為線性獨立
					只要可以找得到就好
				W(x) = 0 表示方法失敗
	線性方程組
		Ax = o 只有零解，稱 trivial solution
		相關計算
			列運算
			高斯消去
			相依、獨立、線性組合
			無解、無限多解、唯一解
	反矩陣
		具左反
			存在 B 使得 BA = I
		具右反
			存在 C 使得 AC = I
		具左反 & 右反
			稱為 invertable, nonsingular
			其反矩陣唯一存在，稱為 inverse
			只可能為方陣
				反過來，方陣下具左右反其一，則可逆
		如何求得
			增廣矩陣後，高斯消去

			列運算
			降階
			遞迴
		矩陣轉置行列式值不變
		det(AB) = det(BA)

		Vandermonde
			以列來看
				由左而右，成等比、且首項為1
			行列式值 = ∏(後 - 前)
				∏ 為連乘
					每個後減前為一項，所有項全部乘在一起
