		生成函數進階處理、求係數
	數列卷積
		an, bn
		求對應的生成函數 A(x), B(x)
		C(x) = A(x) * B(x)
		由 C(x) 求出對應數列 cn
		cn 即為 an, bn 之卷積
	遞迴
		齊次遞迴式求解
			重根
				an = c1 + c2*n + … + ck*n^(k-1)
				重根幾次，多項式次方跟上去幾次
			虛根
				a + bi
				使用極座標，算出角度、半徑
				an = 半徑^n * [c1*cos角度 + c2*sin角度]
		非齊次遞迴式求解
			0. 令通解
				同齊次遞迴式
			1. 令特解
				多項式部分
					多項式次方跟著非齊次項上去
						注意有無特徵根 1
							幾個 1，整個多項式外面就多乘幾個 n
				指數*多項式部分
					跟著非齊次項的指數多項式
					乘上指數項
					乘上指數與特徵根的重根數的 n
				三角函數部分
					c1*cos + c2*sin 就對了
				以上三個部分分開令，加起來就是整個特解
			2. 解特解係數
				特解代回”原遞迴式“
			3. 解通解係數
				通解 + 特解代入初始條件

	生成函數解排列組合
		r 相同物放到 n 相異箱、允許空箱
		對箱子討論
			每個箱子內可以有 0, 1, 2, ... 個物品
			對一個箱子而言
				A’(x) = 1 + x + x^2 + …
				指數為“幾個”、係數為“方法數”
				物品皆相同，方法數都是 1
			對 n 個箱子而言
				A(x) = A’(x)^n
				A(x) 中 x^r 項之係數即所求
		同理
			骰子點數和
			非負整數解
			多重條件的非負整數解
